# NestGenie.uk

Warm, approachable bird-watching products for homes, gardens, and schools.

This repository is a GitHub Pages Jekyll site for the NestGenie placeholder launch.

## Local development

### Prerequisites
- Ruby and Bundler

### Setup

```bash
bundle install
bundle exec jekyll serve --livereload
```

Visit `http://localhost:4000`.

## Current structure

- `index.md` - homepage
- `products.md` - product line placeholder
- `about.md` - brand and mission
- `assets/main.scss` and `_sass/nestgenie.scss` - custom NestGenie visual styling
